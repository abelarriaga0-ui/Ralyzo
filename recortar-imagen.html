¿Cómo recortar una imagen?

  • Selecciona una imagen JPG, PNG o WebP.
  • Marca el área que quieres conservar.
  • Elige recorte libre, cuadrado o circular.
  • Descarga la imagen recortada.

Recortar imagen con forma

Puedes usar recorte cuadrado, horizontal, vertical o circular. El recorte circular se exporta en PNG para conservar el fondo transparente alrededor de la forma.

Recortar imagen con tijera

El recorte funciona como una tijera digital: eliges una zona de la imagen y eliminas todo lo que queda fuera del área seleccionada.

Recortar fotos por el contorno

Esta herramienta permite recortar por área seleccionada. Si buscas recortar exactamente el contorno de una persona u objeto, se necesitaría una función específica de eliminación de fondo.

Consejos para recortar mejor

  • Evita cortar partes importantes del sujeto principal.
  • Usa recorte cuadrado para perfiles o publicaciones 1:1.
  • Usa formato vertical 9:16 para historias o contenido móvil.
  • Usa PNG si necesitas conservar transparencia en un recorte circular.

Recortar imagen online gratis

Con esta herramienta puedes recortar imagen online gratis sin instalar programas. Sirve para eliminar partes innecesarias de una foto, ajustar una imagen para redes sociales, preparar un documento visual o enfocar la atención en una zona específica.

Recortar imagen con forma

Puedes recortar una imagen con forma cuadrada, circular, horizontal o vertical. El modo circular es útil para perfiles, avatares, presentaciones y diseños donde necesitas una imagen con apariencia redonda. Para conservar transparencia, el recorte circular se descarga como PNG.

Recortar imagen online contorno

Muchas personas buscan recortar imagen online por contorno o recortar fotos por el contorno. Es importante diferenciar entre recortar una zona y eliminar el fondo alrededor de una persona u objeto. Esta herramienta recorta el área que tú seleccionas; para separar automáticamente un objeto del fondo se necesita una herramienta específica de recorte por IA.

Recortar imagen en PC o celular

Puedes usar Ralyzo desde PC, tablet o celular. En computadora seleccionas el área con el mouse; en móvil puedes marcar el área usando el dedo. Todo se procesa desde el navegador.

Preguntas frecuentes

  • ¿Puedo recortar imagen gratis? Sí, puedes recortar imágenes sin pagar y sin registro.
  • ¿Funciona con PNG? Sí, acepta PNG, JPG, JPEG y WebP.
  • ¿Puedo recortar con forma circular? Sí, selecciona el modo circular y descarga en PNG.
  • ¿Recorta automáticamente el fondo? No. Esta herramienta recorta la zona seleccionada, pero no elimina fondos automáticamente.
  • ¿Funciona en celular? Sí, puedes seleccionar el área con el dedo en pantallas táctiles.
