¿Cómo girar una imagen?

  • Selecciona una imagen JPG, PNG o WebP.
  • Gírala a la derecha, izquierda o 180°.
  • Aplica espejo horizontal o vertical si lo necesitas.
  • Descarga la imagen final en PNG, JPG o WebP.

Girar imagen espejo

Puedes usar espejo horizontal para voltear la imagen de derecha a izquierda, o espejo vertical para invertirla de arriba hacia abajo.

Girar imagen 360

Puedes usar el control de grados para ajustar la rotación. Si giras 360 grados, la imagen vuelve visualmente a su orientación original.

Corregir orientación de fotos

Esta herramienta es útil cuando una foto aparece acostada, invertida o tomada con mala orientación. Solo debes girarla y descargar la versión corregida.

Consejos para girar imágenes correctamente

  • Usa giro de 90° para corregir fotos verticales u horizontales.
  • Usa giro de 180° cuando la imagen esté completamente invertida.
  • Usa espejo horizontal si necesitas cambiar derecha por izquierda.
  • Usa PNG si quieres conservar transparencia.

Girar imagen online gratis

Con esta herramienta puedes girar imagen online gratis sin instalar programas. Es útil para corregir fotos que quedaron acostadas, girar imágenes escaneadas, ajustar capturas o preparar archivos visuales para documentos, redes sociales y páginas web.

Girar imagen de derecha a izquierda

Si necesitas girar una imagen de derecha a izquierda, puedes usar el botón de giro hacia la izquierda o aplicar espejo horizontal. El giro rota la imagen; el espejo invierte su dirección visual.

Girar imagen espejo

La opción de espejo horizontal cambia la orientación como si la imagen se reflejara en un espejo. También puedes usar espejo vertical para invertir la imagen de arriba hacia abajo.

Girar imagen 360 y rotación personalizada

El control de ángulo permite ajustar la rotación de forma personalizada. Puedes rotar pocos grados para corregir una inclinación o usar giros completos para volver a la orientación original.

Sobre girar imagen 3D

Algunas búsquedas hablan de girar imagen 3D o rotar imagen 3D online. Esta herramienta está enfocada en rotación 2D, giro de fotos, espejo y corrección de orientación. Para efectos 3D reales se necesita un editor avanzado de perspectiva.
